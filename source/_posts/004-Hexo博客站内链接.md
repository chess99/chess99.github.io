# 锚点跳转

在文章A中制作锚点  

```html
<a name="Anchor"/>
```

在文章B中插入跳转

```bash
[文章A](2018/06/11/articleA#Anchor)
```

缺点: 硬编码了YYYY/MM/DD
一种解决方法: 修改`permalink: :year/:month/:day/:title/`为`permalink: :title/`
